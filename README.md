# T Nails Website

This is my first website project, and I made it for my cousin’s nail salon business. The goal was to create a professional and attractive website that helps customers learn about the salon, view services, see photos, and contact the business easily.

## Project Overview

The website was built to showcase the salon’s services, style, and location in a clean and attractive way. It is designed to feel welcoming, elegant, and easy to use on both desktop and mobile devices. The navigation is simple and allows users to click through to different sections of the site smoothly.

## Live Website

You can view the project here:

https://tnailswithington.netlify.app/

## Pages Included

- Home page: welcome section, salon introduction, social links, and booking call-to-action
- Gallery page: image gallery with animated effects and client reviews
- Price page: clear service and pricing layout
- Contact page: address, contact details, opening hours, and embedded map

## Technologies Used

- HTML5 for page structure
- CSS3 for styling, layout, colors, spacing, and responsive design
- JavaScript for interactive navigation, smooth scrolling, and animation effects
- Font Awesome for icons
- Google Fonts for typography
- Bootstrap for styling support
- Google Maps embed for location integration
- Google Form linked to Google Sheets for booking/ enquiry collection instead of a backend at this stage

## What I Learned

During this project, I improved my front-end development skills, especially in:

- CSS Flexbox and layout control
- Creating visually appealing color palettes and design choices
- Building responsive webpages that work on different screen sizes
- Structuring HTML pages clearly and consistently
- Adding interactive elements with JavaScript
- Integrating a Google Map into a website
- Using Google Form with Google Sheets as a simple way to collect customer information
- Using AI tools to help with ideas, code suggestions, and small function improvements
- Learning how AI can improve productivity by speeding up planning, generating design ideas, and helping with simple coding tasks
- Improving prompt engineering by asking clearer questions to get better results from AI
- Using a combination of online tools such as colour suggestion websites and Canva to support the visual design process
- Communication skills to turn a business idea into a clear website plan and ask for the right information from the client

## Design Notes

The website uses a soft, elegant style that fits a beauty and nail salon brand. I focused on creating a polished look with simple navigation, attractive buttons, social media links, and a consistent visual identity.


## What I Want to Improve Later

As I continue developing this project, I would like to:

- Use AI less and rely more on my own problem-solving
- Improve the overall design and make it more professional and unique
- Add a backend for booking appointments and managing enquiries
- Add a more advanced contact/booking system
- Improve accessibility and performance further

## Summary

This project helped me practice building a full-looking website from scratch and understand how design, layout, and user experience work together. It was a strong introduction to creating a polished front-end project for a real business.
